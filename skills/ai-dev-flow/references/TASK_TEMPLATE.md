# 任务模板

## 模板使用说明

A/B 小任务可以只填写必需字段，不要求机械填写完整模板的所有字段。未确认字段写“待确认”，不要编造。

A/B 小任务建议必填字段：

- 任务编号
- 任务标题
- 任务类型
- 当前模式
- 任务状态
- 任务分级
- 背景
- 目标
- 非目标
- 允许修改范围
- 禁止修改范围
- Base commit
- Diff 范围
- 完成标准
- 验证方式
- 修改文件
- Diff 审查
- 用户动作等级 / 验收建议
- 交接摘要

C/D 中大型任务、高风险任务、分支任务和 Worktree 任务应使用完整字段。

> 使用方式：复制本模板到 `docs/tasks/<TASK-ID>.md`，再按项目实际情况填写。未确认的信息写“待确认”，不要编造。
